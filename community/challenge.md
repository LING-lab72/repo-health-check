# 🏆 Repo Health Challenge — 2 周挑战赛

提升你的仓库健康度，赢取社区荣誉！

## 规则

### 时间
- 每 2 周一个周期（周一 00:00 ~ 第二周周日 23:59 UTC）
- 每周一公布上周排名

### 参与方式
1. 访问 [Repo Health Check](http://localhost:5173)
2. 输入你的 GitHub 仓库 URL 进行体检
3. 每分析一次，自动计入排行榜
4. 分享你的 Badge 到社交媒体

### 评分标准

| 等级 | 分数 | 徽章 |
|------|------|------|
| A | ≥ 80 | 🟢 |
| B | ≥ 60 | 🟡 |
| C | ≥ 40 | 🟠 |
| D | < 40 | 🔴 |

### 加分项
- 连续分析同一仓库并提升分数 → +5 额外分
- 在 Issue 中分享改进心得 → +10
- PR 被合并 → +20
- 推荐他人参与 → +5

### 奖励

#### 🥇 冠军
- "最健康仓库" 徽章（永久展示在画廊）
- 仓库在排行榜置顶一周

#### 🥈 亚军
- 社区感谢帖
- 代码 Review 优先权

#### 🥉 季军
- 社区感谢帖

### 每周任务

#### 第 1 周：基础健康
- [ ] 分析你的仓库
- [ ] 完成 code_quality ≥ 80
- [ ] 添加 LICENSE 文件
- [ ] 配置 CI/CD

#### 第 2 周：进阶优化
- [ ] test_coverage ≥ 60
- [ ] 文档完整度 ≥ 75
- [ ] 消除所有 ESLint 错误
- [ ] dependency_security ≥ 80
