# 🏷️ Badge 画廊

展示使用 Repo Health Check 徽章的优秀开源项目。

## 提交你的 Badge

在你的 Issue 中贴上：

```markdown
![health](https://your-domain.com/api/badge/{你的hash})
仓库: https://github.com/owner/repo
描述: 一句话介绍
```

我们会定期更新此画廊。

## 🔗 如何获取 Badge

1. 访问 [Repo Health Check](http://localhost:5173)
2. 输入你的仓库 URL 并分析
3. 在报告页底部找到 Markdown 代码
4. 复制并粘贴到你的 README.md

Badge 会自动显示当前仓库的健康等级，每次有人点击 Badge 链接都会实时更新（基于缓存）。
