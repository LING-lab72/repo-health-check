## 项目概述

Repo Health Check 是一个面向开源项目的仓库健康度体检工具。用户输入 GitHub 仓库 URL 后，系统会克隆并分析仓库，从代码质量、测试覆盖、架构健康、文档完整、依赖安全和工程规范六个维度输出健康度评分、雷达图、问题列表、AI 诊断建议、README Badge、导出报告、排行榜和仓库 PK 对比结果。
